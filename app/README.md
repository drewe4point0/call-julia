# Frontend (Vite + React)

This app starts ElevenLabs voice sessions and renders live transcript UI.

## Environment

Create `.env` from `.env.example`:

- `VITE_ELEVENLABS_AGENT_ID` – your ElevenLabs Conversational AI agent ID
- `VITE_API_BASE_URL` – backend base URL (for `/signed-url` and `/conversation/finalize`)

## Run

```bash
npm install
npm run dev
```
